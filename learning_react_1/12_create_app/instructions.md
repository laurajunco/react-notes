## Ambiente de desarrollo:

1. Instalar create-react-app
```
npm install -g create-react-app
```

2. Ir al directorio donde se va a crear el proyecto

```
create-react-app nombreproyecto
```

3. Ir al directorio del nuevo proyecto

```
cd nombreproyecto
```

4. Iniciar servidor
```
npm start
```
